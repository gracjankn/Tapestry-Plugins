# Tapestry connectors

This repository contains my connector made for the [Tapestry app](https://usetapestry.com) by The Iconfactory. The connector allows you to follow channels from any instance (server) of [PeerTube](https://joinpeertube.org), which is an open, federated video hosting platform, that's based on ActivityPub and is part of the Fediverse. You can think of it as a crossover between YouTube and [Mastodon](https://joinmastodon.org).

## Automatic installation

Click the [Install PeerTube connector](tapestry://add-connector?url=https://git.gracjan.xyz/gracjan/Tapestry-Plugins/releases) link to open it in Tapestry (you need at least Tapestry 1.7 for it to work).

## Manual installation

Alternatively, if you're on an older version of Tapestry or prefer to do it manually, follow these instructions.

1. Head to the [Releases page](https://git.gracjan.xyz/gracjan/Tapestry-Plugins/releases).
2. Download the latest version of the connector.
3. Open Tapestry.
4. Open the app's settings.
5. Go to the **Connectors** tab.
6. Tap **Add from File**.
7. Select the downloaded file.
8. Tap **Open**.
9. Tap **Install Connector**.
